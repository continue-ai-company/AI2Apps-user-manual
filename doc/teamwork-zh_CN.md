## 👥 团队协作

在页面底部工具栏区域，点击`命令行终端`按钮，在弹出框内输入如下命令。
<div><img src="../assets/upload_project1_cn.jpg" alt="upload_project1" /></div>

#### 上传本地项目

<div><img src="../assets/upload_project2_cn.jpg" alt="upload_project2" /></div>

```bash
disk checkin
```

#### 更新本地项目至云端

```bash
disk commit
```

#### 同步云端项目

```bash
disk update
```

#### 添加项目成员

```bash
disk addmb
```
